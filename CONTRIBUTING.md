# Contributing

Terima kasih telah tertarik untuk berkontribusi ke proyek ini! Panduan singkat berikut akan membantu proses kontribusi berjalan lancar.

## Langkah cepat

1. Fork repository ini.
2. Buat branch fitur dari `main` (mis. `feature/nama-fitur`).
3. Buat perubahan di branch Anda.
4. Jalankan dan uji perubahan secara lokal.
5. Buka Pull Request (PR) ke branch `main` dengan deskripsi yang jelas.

## Aturan dasar

- Gunakan branch bernama jelas: `feature/...`, `fix/...`, `docs/...`.
- Sertakan deskripsi perubahan, alasan, dan langkah untuk mereproduksi bila relevan.
- Jika perlu, sertakan screenshot, log, atau contoh output.

## Test & style

- Jalankan test yang ada sebelum membuka PR (jika proyek memiliki test). Jika tidak ada, pastikan fungsi manual dasar berjalan.

## Review

- Reviewer dapat meminta perubahan. Silakan lakukan perubahan pada branch yang sama; PR akan otomatis terupdate.

## Lisensi kontribusi

Dengan mengirimkan PR, Anda setuju kontribusi Anda akan menjadi bagian dari repositori dan mengikuti lisensi proyek.

Jika Anda membutuhkan panduan lebih rinci (conventional commits, pre-commit hooks, CI), beri tahu saya dan saya akan menambahkan contoh konfigurasi.
